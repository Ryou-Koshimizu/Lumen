# 旧URL（/Lumen/）からの転送用

2026-07-15 に GitHub Pages をルート配信へ移行した。
理由: robots.txt / llms.txt はドメインのルートにしか置けない（RFC 9309）。
サブパス配信（/Lumen/）では、書いても一切読まれていなかった。

**本体は https://ryou-koshimizu.github.io/ （リポジトリ: Ryou-Koshimizu.github.io）**

このリポジトリは、旧URLを踏んだ人を本体へ送るためだけのもの。中身を編集しないこと。
noindex を入れているので、検索エンジンには重複コンテンツとして扱われない。
